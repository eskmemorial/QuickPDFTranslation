Quick PDF Translation

Usage

  1. open a PDF file in the internet or local storage
  2. select short text in PDF document
  3. you can get translated text in popup
  4. click any position in PDF document to remove the popup

If selected text consists of more than 100 characters, this extension reject to work.

Privacy Policy

This extension sends selected text to my server to get translated text only when you enable this extension and select text in PDF document. My server stores personal information below for 10 weeks.

  • IP address
  • access time
  • HTTP method
  • HTTP status code
  • number of bytes of response without HTTP header
  • user agent

For more details, read document of NGIX.
