Quick PDF Translation

Setting

Usage

  1. open a PDF file in the internet or local storage
  2. select short text in PDF file
  3. you can get translated text in popup
  4. click any position in PDF file and remove the popup

If selected text consists of more than 100 characters, this extension reject to work.

About translation server

If you suspect that I secretly steal your personal information in my translation server, you have a good choice to avoid outflowing your data. Build translation server by yourself. You can get script of translation server from GitHub.
